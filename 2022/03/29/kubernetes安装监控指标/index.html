kubernetes安装监控指标

发表于2022-03-29
  • github

可能会遇到网络问题，手动创建components.yaml

可能遇到如下问题

  • 1.镜像下载问题

  • 2.启动日志日提示失败

解决办法如下

  • 执行kubectl apply -f components.yaml

  • 执行 kubectl top pod 查看监控情况

  • 刷新dashboard 即可看到监控信息
